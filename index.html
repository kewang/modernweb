Web 2018

  • 臺灣最全方面的 Web 技術盛會

    2 天會期，超過 20 場精彩演講，前端、後端、架構，三大主題面面俱到，精心規劃的議程內容，有趣精彩的場邊活動，來場豐富的 Web 探索之旅吧！

  • 國際作者、開發大師，首度來台！

    著有多本開發書籍的 Jeremy Keith(Clearleft)、專注在 AMP 的 Paul Bakaus(Google)、吃太多糖果的 Felix Rieseberg(Slack)，多位大師首度來台分享 Web 技術。

  • 中國最新 Web 技術實務經驗交流

    二大出品人領軍，特別策劃前端、後端架構的議程，帶來第一手的中國 Web 技術觀點，與實務心得經驗。

Modern Web 2018

  • 1

    一想到網站技術大會，就會想到 ModernWeb

    連續 3 年大受好評，2018 年 為期 2 天的會期，500 人與會，超過 20 場專業演講，精彩的議程內容，更多大師登場，前端、後端、架構，三大主題全面展開，來自中國的 Web 技術分享 - 特別策劃一天的中國技術日。還有滿滿的場邊活動，將再次打造全新的「心」感受。

  • 2

    首度登台！國際大師坐了一天飛機也要來！

    你一定都有看過他寫的書，同時也是 Web 開發者的 Jeremy Keith，首度從英國遠渡重洋來台，帶來最新的觀點，最精彩的演說，錯過這一次，可能要再等 4 年；來自於美國 Slack 的 Felix Rieseberg ，喜歡吃糖果的大男孩，投入在 Open Source 的懷抱中，將在這首次登上的舞台，帶來豐富的議程演說。

  • 3

    電腦不會挑選議程，但是我們會！

    大會的的議程 200% 都是由專業團隊嚴選，今年 ModernWeb18 講者投稿熱烈，在眾多投稿中仔細審核內容、反覆討論而定案。為滿足更多 Web 人的需求，2018 年，特別規劃，從前端、後端及架構三大主題，滿滿的 2 天，滿滿的收獲。

  • 4

    前端巧手、後端好手、全端高手，全台好手齊聚！

    參加研討會，除了專心聽議程吸收經驗之外，最重要的當然是把握現場與其它好手交流、還有與專家面對面討教的機會。在這個場域中，將聚集了全台的各路好手，不論是技術交流、認識朋友、吸收知識或是獲取經驗，都是非常難能可得寶貴的機會。

  • 5

    好吃、好玩、又好拿！滿足你的腦子與肚子

    研究指出，大腦就算「甚麼也不想」，每分鐘也會消耗0.1卡，若是在思考較複雜的事情時，每分鐘甚至會燃燒1~1.5卡，為了維持大腦正常的運作，將供應各種補充能量的「補給品」，累了嗎？補充能量再上；當然還有不能錯過的 ModernWeb 18 大會紀念服等你來收集！

  • 里程碑

  • 2018 July

    三大主題議程，從前端、後端、架構，開拓更廣闊的 WEB 新視野。世界級技術叢書作者，同時也是知名開發者 Jeremy Keith，首度來台。

  • 2017 July

    議程大爆滿，超豐富的二天四軌議程，首度超過 30 場精彩的專業講座。來自世界各地的大師， Google、Slack、騰訊 …. 最多國際大師的場域。

  • 2016 Aug

    首次精心策劃，中國電商 Web - 中國技術日，廣邀中國專家登台。台灣 Web 開發者年度票選，再參加期待度最高的研討會。

  • 2015 May

    首度在台舉辦，吸引超過600人報名，500人與會，大受好評。Web 人不能不認識的 Java Scrip 之父、PHP 之父，重量級登場！
